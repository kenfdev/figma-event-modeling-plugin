# F7.1: KeyboardShortcuts

## Type
Infrastructure

## Description
Cmd/Ctrl+Shift+Letter triggers element creation.

## User Story
As a user, I want to use keyboard shortcuts to quickly create elements without clicking buttons.

## Acceptance Criteria
- [ ] Shortcuts only work when FigJam canvas is focused
- [ ] Shortcuts are disabled when typing in text inputs
- [ ] Modifier pattern: Cmd/Ctrl + Shift + Letter
- [ ] Shortcut mappings:
  | Element | Shortcut |
  |---------|----------|
  | Command | Cmd+Shift+C |
  | Event | Cmd+Shift+E |
  | Query | Cmd+Shift+Q |
  | Actor | Cmd+Shift+A |
  | Lane | Cmd+Shift+L |
  | Chapter | Cmd+Shift+H |
  | Processor | Cmd+Shift+P |
  | Screen | Cmd+Shift+S |
  | Slice | Cmd+Shift+I |
  | GWT | Cmd+Shift+G |
- [ ] Each shortcut triggers the corresponding create command

## Dependencies
- F1.1-F1.4: Core element creation
- F4.1-F4.4: Structural element creation
- F5.1-F5.2: Section element creation

## Technical Notes
- Must detect whether canvas or input field is focused
- Cmd (Mac) / Ctrl (Windows/Linux) modifier handling
